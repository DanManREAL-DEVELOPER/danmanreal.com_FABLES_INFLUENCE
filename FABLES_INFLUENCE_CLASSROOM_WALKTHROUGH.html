The project is a harness factory. It does not merely say "work harder." It builds a playbook, ledgers, wargames, status truth, and a verifier so the next agent knows exactly what to do.

What is FABLES_INFLUENCE?

10-year-old answer

It is a kit that helps an AI make a project finishable. It creates the instructions, checklists, notebooks, safety rules, and proof checks so work does not get lost.

Teacher note

Start with the backpack analogy: the repo is not the finished school project. It is the backpack that gives the next AI all the tools to finish a project safely and prove it.

Use case

Use it when a project is too big for one chat, has many agents, needs owner decisions, or needs evidence before anyone says "done."

IV. What gets generated

FABLES_HARNESS

When the genesis prompt runs in a target project, the AI builds a harness folder. These are the durable objects the class should recognize.

V. Use cases

When Do We Use It?

FABLES_INFLUENCE is project-agnostic. That means the same method can help many kinds of work, from a game to a school club website.

VI. Teacher board

Class Plan

Use these tiles as the live lesson order. Each tile includes a short classroom script and the common question it answers.

Teacher script you can say out loud

"Imagine a robot helper. If you only say 'finish my project,' it may guess. FABLES_INFLUENCE gives the robot a map, a checklist, and a proof test."
"In this system, done is not a feeling. Done means the file exists, the ledger says where it is, and the verifier can check it."
"Some questions belong to the human owner. The AI is allowed to stop and ask instead of inventing the answer."
"Before the mission begins, the system plays a pretend battle against mistakes. That is called a wargame."
